# TodoApp
Django学習用個人開発

---
## 🚀使用技術
- **フロントエンド**：HTML, CSS, Bootstrap
- **バックエンド**：Python（Django）, MySQL
- **インフラ**：インフラ

---
## ▶️  起動方法
```bash
# 環境変数ファイルの作成
cp .env.example .env

# ビルド
docker comose build

# 起動
docker compose up

# ブラウザで確認
http://localhost:8000/login/
```
